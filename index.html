ABOUT

Becoming My Childhood Hero

A quick elevator pitch: From a pretty young age, I've found myself drawn to technology. Like most 5-year-olds, I thought I was going to grow up to become a scientist, an astronaut, and a wizzard. Unlike most kids, however, I'm still chasing those dreams. My end goal? Use technology to change the world.

Registration Form

Come One, Come All!

Often times, developers are thrown into a project late in the game, and asked to finish up a job someone else started. Here, I was asked to hook up a website built by others to a MailChimp campaign that I created through embedded registration forms. I quickly learned about the MailChimp API, how to use SCSS, and embedding forms with iframes.

Database Design

Good Ole Adobe

Did I mention that I'm also a student? Pictured here is a relational graph for a database designed by me and two other students for a Database Design class last year. We studied entity and attribute relations, data clustering, data mining, and even touched on the newer ideas of Big Data. Sure, the project was all about Pokemon, but that didn't make it any easier!

WordPress Plugin

Battling the Multisite

One of my first projects while working at the Digital Corps last year was to create a plugin for a Ball State WP Multisite, which allowed network admins of the multisite to publish and manage videos from Vimeo and Lynda for the users of the multisite to watch. This was my first time using WP, but have since gone on to create plenty of WP sites and plugins.
